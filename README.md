## Как работает алгоритм

Шифр Билайна — это перестановочный шифр.

Текст записывается "змейкой" (зигзагом) по строкам:

```
H   O   L
 E L W R D
  L   O
```

Исходный текст:

```
HELLOWORLD
```

Результат:

```
HOLELWRDLO
```
