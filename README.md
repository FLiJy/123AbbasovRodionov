# Отладка приложения «Фибоначчи»

## Описание
Консольное приложение для вычисления чисел Фибоначчи.  
В исходном коде присутствовала ошибка в условии цикла, из-за которой программа возвращала неверный результат.

## Исправления
- Исправлено условие цикла: `for (int i = 2; i < n; i++)` → `for (int i = 2; i <= n; i++)`
- Добавлены документирующие XML-комментарии к методам и классу.

## Способы отладки Microsoft Visual Studio, которые были использованы:

- Установка обычных **точек останова** (Breakpoints)
- Пошаговое выполнение:
  - **F10** — Шаг с обходом (Step Over)
  - **F11** — Шаг с заходом (Step Into)
  - **Shift + F11** — Выход из метода (Step Out)
- Окна отладки:
  - **Locals** (локальные переменные)
  - **Autos** (автоматические переменные)
  - **Watch** (контрольные значения)
- **Data Tips** — подсказки по данным при наведении мыши на переменные
- Изменение значения переменных во время отладки
- Просмотр и анализ стека вызовов (**Call Stack**)
- Запуск приложения в режиме отладки (Debug)

## Результат
После исправления ошибка устранена. При вызове `Fibonacci(5)` программа корректно выводит результат **5**.
