**Цель работы**

Провести тестирование программных модулей авторизации и регистрации с использованием автоматизированных Unit-тестов в Microsoft Visual Studio.
 
**Выполнение работы**

В рамках работы был создан отдельный тестовый проект PR14.Tests, в котором реализованы автоматизированные тесты для проверки:
•	модуля авторизации
•	модуля регистрации

Перед созданием тестов был проведён рефакторинг кода:
•	логика авторизации вынесена в метод Auth
•	логика регистрации вынесена в метод Register

Это позволило тестировать функционал независимо от пользовательского интерфейса.
 
Тестирование авторизации

**Были реализованы следующие тесты:**

1. AuthTest

Проверка авторизации с некорректными данными.
Ожидаемый результат: отказ в доступе.

2. AuthTestSuccess

Проверка авторизации с корректными данными пользователя из базы данных.
Ожидаемый результат: успешный вход.

3. AuthTestFail

Проверка различных негативных сценариев:
•	пустые поля
•	неверный логин
•	неверный пароль

Ожидаемый результат: авторизация не выполняется.
 
Тестирование регистрации

Были реализованы тесты:

1. RegisterTestSuccess

Проверка успешной регистрации нового пользователя.

2. RegisterTestFail

Проверка ошибок регистрации:
•	пустые поля
•	попытка создать пользователя с существующим логином
