 📐 123AbbasovRodionov

> Десктопное приложение на WPF для выполнения математических вычислений

  Описание

123AbbasovRodionov — это учебное приложение на C с использованием технологии Windows Presentation Foundation (WPF), предназначенное для расчёта математических выражений по заданным формулам. Приложение разработано в рамках учебного проекта (вариант 12).

🔧 Основные возможности

- Навигация между страницами через боковое меню
- Расчёт математической формулы (вариант 12, формула 9.b):
  
  ```
  f = (|x-y| * (sin²(z) + tg(z))) / (y * (x+1))
  ```
- Валидация пользовательского ввода
- Обработка ошибок (деление на ноль, некорректный формат чисел)
- Подтверждение при выходе из приложения
- Очистка полей ввода одной кнопкой

  Требования

- ОС: Windows 10/11
- .NET Framework: 4.7.2 или выше
- IDE: Visual Studio 2019/2022 (рекомендуется)

  Установка и запуск

 Через Visual Studio

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/FLiJy/123AbbasovRodionov.git
   ```

2. Откройте решение `123AbbasovRodionov.slnx` в Visual Studio

3. Восстановите NuGet-пакеты (автоматически при сборке)

4. Нажмите `F5` или кнопку Start для запуска

 Сборка из командной строки

```
 Перейдите в папку проекта
cd 123AbbasovRodionov/123AbbasovRodionov

 Сборка через MSBuild
msbuild 123AbbasovRodionov.csproj /p:Configuration=Release
```

 📖 Использование

1. Запустите приложение
2. Введите значения переменных x, y, z в соответствующие поля
3. Нажмите «Рассчитать» для получения результата
4. Результат отобразится с точностью до 6 знаков после запятой
5. При ошибке ввода появится сообщение с описанием проблемы
6. Кнопка «Очистить» сбросит все поля

>  Важно: Убедитесь, что `y ≠ 0` и `x ≠ -1`, чтобы избежать деления на ноль.
