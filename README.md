> Десктопное приложение на WPF для выполнения математических вычислений

Описание

Этот проект - это учебное приложение на C с использованием технологии Windows Presentation Foundation (WPF), предназначенное для расчёта математических выражений по заданным формулам для варианта 12.

🔧 Основные возможности

- Навигация между страницами через боковое меню
- Расчёт математических формул
- Валидация пользовательского ввода
- Обработка ошибок (деление на ноль, некорректный формат чисел)
- Подтверждение при выходе из приложения
- Очистка полей ввода одной кнопкой

 📖 Использование

1. Запустите приложение
2. Введите значения переменных x, y, z в соответствующие поля
3. Нажмите «Вычислить» для получения результата
4. Результат отобразится с точностью до 6 знаков после запятой
5. При ошибке ввода появится сообщение с описанием проблемы
6. Кнопка «Очистить» сбросит все поля

>  Важно: Убедитесь, что `y ≠ 0` и `x ≠ -1`, чтобы избежать деления на ноль.
